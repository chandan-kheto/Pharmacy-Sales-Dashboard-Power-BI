# 📊 Pharmacy Sales Dashboard (Power BI)

## 📝 Project Overview

This project focuses on analyzing **pharmacy OTC sales data** using an interactive Power BI dashboard.
The goal is to extract meaningful business insights from raw data and support data-driven decision-making.

---

## 🎯 Objectives

* Analyze sales performance across different **countries and products**
* Track **monthly sales trends**
* Identify **top-performing products and regions**
* Build an **interactive dashboard** for dynamic filtering

---

## 📊 Dashboard Features

### 🔹 Key KPIs

* **Total Sales ($)**
* **Total Units Sold**

### 🔹 Visualizations

* **Sales by Country** (Bar Chart)
* **Sales by Product** (Bar Chart)
* **Monthly Sales Trend** (Line Chart)

### 🔹 Interactivity

* **Country Filter (Slicer)** to dynamically update all visuals

---

## 📈 Key Insights

* USA generates the highest revenue among all countries
* Digestive Enzyme is the top-performing product
* Sales show fluctuations with a dip in March and peak mid-year
